## ✨ Features

### 📋 Task Manager
Quickly add, check off, and delete tasks. Designed to be frictionless — because with ADHD, starting is the hardest part.

### ⏱️ Pomodoro Focus Timer
A built-in 25-minute focus timer with Start, Pause, and Reset controls. Work in focused sprints so your brain doesn't have to fight against time.

### 😊 Mood Tracker
Log how you're feeling throughout the day with one tap. Track your emotional patterns over time with a simple emoji-based system.

### 📝 Quick Notes
A distraction-free notepad for capturing ideas before they vanish. Notes are saved locally so they persist between sessions.

## 🚀 Getting Started

No installation needed! This is a pure HTML/CSS/JS app — just open the file in any browser.

### Option 1: Clone & Open Locally

```bash
# Clone the repository
git clone https://github.com/YOUR-USERNAME/adhd-assistant.git

# Navigate to the project folder
cd adhd-assistant

# Open in your browser
open index.html
# or on Windows:
start index.html
```

### Option 2: Use the Live App

👉 Visit [https://zupe.app/adhd-assist](https://zupe.app/adhd-assist) directly — no setup required.

## 🎯 Who Is This For?

This app was built with love for anyone who:

- 🌀 Struggles to keep track of tasks throughout the day
- 🔥 Needs help entering and staying in a "flow state"
- 📉 Wants to notice patterns between mood and productivity
- 💭 Has thoughts that vanish before they can be written down

You don't need to have an ADHD diagnosis to benefit — this is for **anyone who wants a calmer, simpler productivity tool.**
